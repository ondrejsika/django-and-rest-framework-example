## Install

```
git clone git@github.com:ondrejsika/django-and-rest-framework-example.git
cd django-and-rest-framework-example
python3 -m venv venv
. venv/bin/activate
pip install pipenv
pipenv install
./manage.py migrate
```

## Create super user

```
./manage.py createsuperuser
```

## Run

```
./manage.py runserver
```
